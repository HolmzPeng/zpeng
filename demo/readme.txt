本目录所有代码运行依赖条件:
    安装sdl库:
        sudo apt install libsdl1.2-dev libsdl-mixer1.2-dev -y
        sudo apt install libsdl-image1.2-dev libsdl-ttf2.0-dev -y

    重新编译:
        进入目录运行:
            make clean
            make
